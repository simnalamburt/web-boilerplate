My Web Boilerplate
========

To change default appname

```
ag appname
```

### Client

```sh
# prerequisite
npm install bower webpack-dev-server -g

cd client
npm install && bower install
webpack-dev-server --progress --colors
```

### Server

```sh
# prerequisite
gem install rails --no-document

cd server
bundle install
rails server
```
