My Web Boilerplate
========

```
# To find where default appname has been used
ag appname

# Prerequisite
npm install bower webpack-dev-server -g
gem install rails --no-document

# Dependencies
npm install
bower install
bundle install

# Server
webpack-dev-server --progress --colors
cd server && rails server
```
